### Steps to Make a Commit in Git
1. **Check file status:** Run `git status` to see your modified or untracked files.
2. **Stage the changes:** Run `git add <filename>` (or `git add .` to stage all changes) to prepare files for the commit.
3. **Commit the changes:** Run `git commit -m "Your descriptive commit message"` to permanently save your staged snapshot.
4. **Push to GitHub (Optional but recommended):** Run `git push origin main` to send your local commits to your remote repository.
