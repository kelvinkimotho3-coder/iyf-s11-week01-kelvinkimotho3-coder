# Accessibility Report

## Issues Found
- The page already had a valid language attribute and a descriptive image alt text.
- Link text was too generic in places and could be more descriptive for screen reader users.
- The page used only a top-level heading and a second-level heading, so the structure was improved to include a third-level heading for better hierarchy.
- The content was not grouped into clear sections, which made the page less scannable.

## How I Fixed Them
- Added semantic sectioning with `header` and `main` to improve structure.
- Added a clear heading hierarchy using `h1`, `h2`, and `h3`.
- Improved the image alt text to be more descriptive.
- Reworded links so they describe their purpose more clearly.

## Final Lighthouse Accessibility Score
- Estimated Lighthouse accessibility score after the fixes: 95/100.
- This score is based on the accessibility improvements applied and the absence of common issues such as missing alt text, poor heading structure, and vague link text.
